【社会服务赛道】巴威 BAVI — 台风实时路径监测与防灾预警平台

【标签】 社会服务 社会公益

---

## 1. 创意名称 + 创意介绍

**创意名称**：巴威 BAVI — 台风实时路径监测与防灾预警平台

**想解决什么问题**：台风来袭时，公众往往只能通过零散的新闻报道或社交媒体的二次转述获取台风信息，信息来源混杂、时效性差，缺少一个直观、实时、权威的可视化平台来追踪台风路径与强度变化。

**为什么会想到做这个**：今年第 9 号台风「巴威」生成后，我在各大平台搜索台风路径图时发现，大部分信息都是静态图片或延迟严重的页面。于是我想到：能不能直接从日本气象厅（JMA）的公开数据接口拉取实时数据，做一个真正实时、无延迟的台风监测页面？

**大概是什么产品**：一个纯 HTML 单页面的台风监测网站，从日本气象厅公开 API 实时获取台风位置、气压、风速、路径预报与风圈数据，通过自绘地图直观展示，支持 5 分钟自动刷新。

---

## 2. 目标用户及痛点

**面向哪些用户**：
- 沿海城市居民（需要提前了解台风动态，做好防灾准备）
- 气象爱好者（关注台风路径、强度变化等专业数据）
- 应急管理人员（需要实时数据辅助决策）
- 船员、渔民、沿海从业者（生命财产安全高度依赖台风预报）

**在什么场景下使用**：
- 台风季来临前后，用户随时打开网页查看台风当前位置与未来 120 小时路径预报
- 出行前查询目的地是否在台风影响范围内
- 了解台风的中心气压、最大风速、风圈半径等关键指标，判断风险等级

**当前痛点**：
- 官方气象网站多为静态图片，更新不及时
- 专业气象数据（如 JMA 公报）以原始格式呈现，普通用户难以理解
- 地图依赖国外瓦片服务，国内加载缓慢或无法访问
- 缺乏将实时数据、路径可视化、防灾提示整合在一个页面的轻量工具

---

## 3. 价值与意义

**社会价值**：
- 降低公众获取台风实时信息的门槛——打开一个网页就能看到台风在哪、往哪走、有多强
- 通过集成应急管理部、中央气象台、国家预警发布等官方入口，引导用户获取权威预警
- 内置防灾提示（关注预警、远离危险、居家准备、紧急联络），在关键时刻可能挽救生命
- 地图完全自绘（Canvas 矢量渲染），不依赖 Google Maps 等被墙服务，中国境内任何网络环境均可流畅访问

**技术创新**：
- 数据源直接对接日本气象厅 JMA 公开 API，5 分钟自动刷新，零延迟
- 自绘 Canvas 地图：包含中国大陆、日本、朝鲜半岛、菲律宾等 10 组海岸线 + 25 个城市标记 + 经纬度网格
- 风圈可视化：30kt 强风圈 + 50kt 暴风圈以椭圆渲染，直观展示台风影响范围
- 概率圈渲染：未来各预报点的 70% 概率圈以虚线椭圆展示
- 实况点脉冲动画：当前台风位置带辐射光晕 + 脉冲环 SVG 动画，一眼可见
- 无需服务器、无需 API Key、无需数据库，一个 HTML 文件即可运行
