# vibecoding-claude

基于 Claude Code 的 vibecoding 项目。

## 环境配置

本项目使用 DeepSeek API 作为 Anthropic 兼容后端。

- **默认模型**: `deepseek-v4-pro[1m]`
- **轻量模型**: `deepseek-v4-flash`
- **Effort Level**: `max`

## 开始使用

1. 使用 VS Code 打开 `claude.code-workspace`
2. 启动 Claude Code 开始交互式开发

## 项目结构

```
claude/
├── .claude/
│   └── settings.json      # Claude Code 配置
├── claude.code-workspace   # VS Code 工作区
└── README.md
```
